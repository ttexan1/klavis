## Getting started

Connect to Google Sheets to create, read, and manage spreadsheets and automate data workflows through AI agents.

<Tabs>
  <Tab title="API">
    <Steps>
      <Step title="Install the SDKs (optional)">
        <CodeGroup>
          ```bash pip
          pip install klavis
          ```
          ```bash npm
          npm install klavis
          ```
        </CodeGroup>
      </Step>
      
      <Step title="Create a Strata MCP Server with Google Sheets">
        <CodeGroup>
          ```python Python
          from klavis import Klavis
          from klavis.types import McpServerName
          
          klavis_client = Klavis(api_key="YOUR_API_KEY")
          
          response = klavis_client.mcp_server.create_strata_server(
              servers=[McpServerName.GOOGLE_SHEETS],
              user_id="user123"
          )
          ```
          
          ```typescript TypeScript
          import { Klavis } from 'klavis';
          
          const klavis = new Klavis({ apiKey: 'YOUR_API_KEY' });
          
          // Create a Strata MCP server with Google Sheets
          const response = await klavis.mcpServer.createStrataServer({
              servers: [Klavis.McpServerName.GoogleSheets],
              userId: "user123"
          });
          ```
          
          ```bash cURL
          curl -X POST "https://api.klavis.ai/mcp-server/strata/create" \
            -H "Authorization: Bearer YOUR_API_KEY" \
            -H "Content-Type: application/json" \
            -d '{
              "serverName": "Google Sheets",
              "userId": "user123"
            }'
          ```
        </CodeGroup>

        <Card title="API Reference" icon="magnifying-glass" href="/api-reference/strata/create" horizontal>
          Full Strata API endpoints
        </Card>
      </Step>
      
      <Step title="Authenticate">
        <CodeGroup>
          ```python Python
          import webbrowser
          
          # Open OAuth authorization page
          webbrowser.open(response.oauth_urls[McpServerName.GOOGLE_SHEETS])
          ```
          
          ```typescript TypeScript
          import open from 'open';
          
          // Open OAuth authorization page
          await open(response.oauthUrls[Klavis.McpServerName.GoogleSheets]);
          ```
          
          ```bash cURL
          # Copy and paste the OAuth URL into your browser
          echo "Visit this URL to authorize: https://api.klavis.ai/oauth/gsheets/authorize?instance_id=YOUR_INSTANCE_ID"
          ```
        </CodeGroup>

      </Step>

## Available Tools

<Accordion title="Tools Information">
| Tool Name | Description |
|-----------|-------------|
| google_sheets_create_spreadsheet | Create a new spreadsheet with a title and optional data |
| google_sheets_get_spreadsheet | Retrieve spreadsheet properties and cell data for all sheets (‚ö†Ô∏è May cause context limits with large spreadsheets) |
| google_sheets_list_sheets | List all sheets in a spreadsheet (metadata only, no cell data) - **Recommended for discovery** |
| google_sheets_get_sheet_by_name | Get cell data for a specific sheet by name - **Recommended over get_spreadsheet** |
| google_sheets_get_range | Get cell data for a specific range using A1 notation (e.g., 'Sheet1!A1:D10') - **Most efficient** |
| google_sheets_write_to_cell | Write a value to a specific cell in a spreadsheet |
| google_sheets_list_all_sheets | List all Google Sheets spreadsheets in the user's Google Drive |
</Accordion>

<Note>
**Best Practices for Large Spreadsheets:**
1. Use `google_sheets_list_sheets` first to discover available sheets
2. Use `google_sheets_get_range` to fetch only the data you need
3. Use `google_sheets_get_sheet_by_name` when you need a full sheet
4. Avoid `google_sheets_get_spreadsheet` for spreadsheets with multiple large sheets
</Note>

<Note>For more details about tool input schema, use the [get_tools](https://www.klavis.ai/docs/api-reference/mcp-server/get-tools) API.</Note>
